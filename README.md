# JustFriends (FYI)

### Setup Environment
Make sure that [Django](https://github.com/django/django) is installed.

You either do this by following the instructions through the above link, or by running:
```bash
    pip install django
```


### Start Project
```bash
    python manage.py runserver 8080
```

### Run UnitTests
```bash
    python tests.py
```
